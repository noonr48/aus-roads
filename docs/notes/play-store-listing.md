## App name
aus-roads: Offline Australia Map

## Short description
Privacy-focused offline map with live traffic, routing, and navigation for South Australia.

## Full description
aus-roads is a privacy-focused offline map application for South Australia (with NSW and Victoria support coming soon).

**Features:**
• Offline map — no internet required for map viewing
• Free-text search — find suburbs, roads, POIs instantly
• Live traffic — official roadworks, incidents, closures from Traffic SA
• Outback road warnings — DIT road conditions for remote areas
• Offline routing — turn-by-turn navigation powered by Valhalla
• Voice prompts — TTS turn instructions in Australian English
• Route history — recent routes stored locally
• Pin drops — mark and organize locations with colors
• Multi-theme — System/Light/Dark mode support
• Tablet layout — optimized for larger screens

**Privacy:**
• No internet permission (offline flavor)
• No location tracking (until you start navigation)
• No analytics, no ads, no accounts
• All data stored locally on your device
• Open source (AGPL-3.0)

**Data sources:**
